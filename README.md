# CodeAlpha Stock Trading Platform

## Overview
A Java-based Stock Trading Platform Simulator developed as part of the CodeAlpha Internship Program. The project demonstrates Object-Oriented Programming (OOP) concepts by simulating stock trading operations.

## Features
- View Market Data
- Buy Stocks
- Sell Stocks
- Portfolio Management
- Transaction History
- Balance Tracking
- File Handling Support

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- HashMap
- File Handling

## Project Structure
- Main.java
- Market.java
- Stock.java
- User.java
- Portfolio.java
- Transaction.java

## How to Run
1. Compile all Java files.
2. Run Main.java.
3. Use the menu options to perform stock trading operations.
